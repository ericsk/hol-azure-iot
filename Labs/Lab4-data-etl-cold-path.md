# Lab 4: 如何處理大量資料

這份 Lab 希望讓學員瞭解，如何處理儲存在儲存體中的資料，此為一般 data pipeline 中的 cold path。

## 1. 環境準備

* Azure HDInsight 叢集。

## 2. Hands-on Tasks

* 將儲存在 storage 裡的資料載入 HDInsight 中 (Hive table or SparkQL 皆可)。

* 使用 Excel/Power BI 等相關工具載入由 HDInsight 處理完後的資料。
